What is Code for Palestine "WeCODE"?

We code is a program which aims to provide Palestinian students with a Smarter Future,and give the students a strong skill-set in programming, business, design thinking, leadership, team work, and entrepreneurial skills. Students will participate in the program for three years. During each year, the program will include a two-week summer camp as well as year-long weekly sessions. This year's summer camp includes students from all over Palestine including Gaza and instructors from Stanford University and a local CS instructor. In two weeks, students will learn different types of programming skills each year.
